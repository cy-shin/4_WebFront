레이아웃(layout)

        사전적 의미 : 배치, 정리
        기술적 의미 : 구성 요소(html요소)를 제한된 공간(body태그 영역)에 효율적으로 배치하는 것

화면 배치 방법(형식) : display 속성 

        요소가 화면에 어떻게 보여질지 형식을 지정하는 속성

        1. block
        - 화면을 수직 분할(행을 나눔)
        - 한 행을 전부 차지함(사용하면 줄바꿈이 됨)
        - width / height 사용 가능

        2. inline
        - 화면을 수평 분할(열을 나눔)
        - 작성한 영역만큼만 차지
        - width / height 사용 불가능

        3. inline - block
        - inline의 수평 분할 + block의 크기 조절

        4. none
        - 화면에 요소가 표시되지 않으나, 존재는 하고 있는 상태

        5. * flex
        - 요소의 정렬되는 방향, 간격을 유연하게 처리하는 형식
