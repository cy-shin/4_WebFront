1. 목록 관련 태그 

        ul(Unordered List) 태그 : 순서가 없는 목록

        ol(Ordered List) 태그 : 순서가 있는 목록(1, 2, 3, ...)

        li(List Item) 태그 : ul, ol 태그 내부 목록의 요소를 나타내는 태그

(1) ul 태그

    내용을 바로 적으면 목록으로 인식되지 않음
    li 태그로 감싸야 목록으로 인식됨
  • 김밥
  • 서브웨이
  • 백반
  • 햄버거

(2) ol 태그 

        ol 태그의 type 속성

        type = "a" -> 소문자 알파벳을 사용
        type = "A" -> 대문자 알파벳을 사용
        type = "i" -> 소문자 로마 숫자를 사용
        type = "I" -> 대문자 로마 숫자를 사용


        * start 속성 : 시작 번호 지정

        * reversed 속성 : 역순
    내용을 바로 적으면 목록으로 인식되지 않음
    li 태그로 감싸야 목록으로 인식됨

  4. 햄버거
