영역 관련 태그 


        
 block 형식(성질) 

        - 공간을 수직 분할 (한 줄을 차지함)
        -> 수직으로 되어있는 화면에 가로선을 그어 여러 행(row)으로 분할(행?)

        - 크기를 지정하는 css속성인 width, height를 사용할 수 있다

        ex) p / pre (문단), h1 ~ h6(제목 영역 나누기),
            hr(수평선을 넣어서 화면 나누기),
            div(영역 나누기)


        
 inline 형식(성질) 

        - 공간을 수평 분할 (한 칸을 차지함)
        -> 수평으로 되어있는 화면에 세로선을 그어 여러 열(column)로 분할

        - 크기를 지정하는 css속성인 width, height를 사용할 수 없다

        ex) strong/b (영역 내 글자를 굵게),
            em/i(영역 내 글자를 기울이기),
            mark(글자에 노란색 배경),
            span(한 줄 내에서 열을 나누는 영역)

block / inline 차이점 1 : 줄바꿈(영역 분할 방식)

block(수직 분할)

첫 번째 영역
두 번째 영역
세 번째 영역
        block 형식 태그는 한 줄을 모두 차지하기 때문에
        뒤에 작성되는 내용은 다음 줄에 출력된다.

inline(수평 분할)

첫 번째 영역두 번째 영역 세 번째 영역 
        inline 형식 태그는 한 줄 내에서
        작성된 내용 만큼의 칸(열)을 차지하기 때문에
        다음에 작성된 내용이 줄을 바꾸지 않고 뒤에 붙어서 출력된다.

       
        * html문서에서 inline 형식의 태그 성 중
        태그 앞이나 뒤에 엔터를 입력해서 작성하는 경우
        출력되는 화면에서 한칸 띄어쓰기 된 형태로 출력됨
