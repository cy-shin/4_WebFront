Node와 Element의 차이

        Node : 태그(요소 노드), 속성, 주석, 내용(텍스트 노드) 등을 모두 표현.

        Element : Node의 하위 개념으로 요소 노드만을 표현


        [Element만 탐색하는 방법] 

        children : 자식 요소만 모두 선택
        parentElement : 부모 요소 선택

        firstElementChild : 첫 번째 자식 요소 선택
        lastElementChild  : 마지막 자식 요소 선택

        previousElementSibling : 이전 형제 요소 선택
        nextElementSibling     : 다음 형제 요소 선택

요소 생성 및 추가

innerHTML을 이용한 요소 생성 및 추가

        * 문제점!
        1) 특정 요소의 내부 내용을 새로운 내용으로 덮어씌움
           (기존 내용이 사라지는 문제 발생)
           기존 관련 HTML 코드를 지우고, 해당 위치에 새 HTML 코드를 만들어 덮어 씌우는 방식으로 처리됨
        예) 박스를 추가 시
        1-1) box1 추가
        1-2) box2 추가 시 -> 기존 box1을 지우고 다시 box1과 box2를 만들어서 덮어씌움
        1-3) box3 추가 시 -> 기존 box1, box2를 지우고 box1, box2, box3을 만들어 덮어씌움
        -> 이때, 2)의 box2와 3)의 box2는 다른 요소!

        2) 덮어씌워지는 새로운 내용을 단순 문자열로 추가하는게 아니라,
        HTML 요소가 포함되어 있으면 이를 해석할 수 있게 HTMLParser를 이용함
        추가적인 해석을 진행하므로 속도가 느림

        -> innerHTML말고 document.createElement()를 사용할 것
